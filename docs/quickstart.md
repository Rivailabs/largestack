# Quick Start

## Install

```bash
pip install largestack
export LARGESTACK_OPENAI_API_KEY=<openai-api-key>
```

## Scaffold a project

```bash
largestack new my-agent
cd my-agent
pip install -e .
python main.py
```

## Develop with hot-reload

```bash
largestack dev
# Opens http://localhost:4111
```

# Largestack AI POC offer

## Offer

10–15 day Secure AI Document/RAG Assistant POC.

## Public pricing language

Paid POCs start from ₹2L depending on scope, data, deployment, integrations, and security requirements.

Do not present ₹2L–₹5L as a hard fixed price on the homepage. Use “starting from ₹2L”.

## Typical POC deliverables

- Document ingestion
- RAG question answering
- Citations/sources
- Guardrails
- Logs/traces
- Simple demo UI
- Docker/local/cloud demo
- Final report and production roadmap

## Ownership

- Largestack framework remains owned by RivaiLabs.
- Client data remains owned by the client.
- Client-specific workflows/documents remain confidential.
- Generic reusable templates, connectors, and improvements remain reusable by RivaiLabs unless otherwise agreed in writing.

# Examples and Use Cases

Largestack was validated not only with unit tests, but also with generated mini-products and business-style scenarios.

These are **working mini-product slices**, not full SaaS products with complete UI, billing, auth, and production operations.

## Recommended public demos

Do not demo all 50. Use three high-signal demos:

| Demo | Why it works |
|---|---|
| Support Ticket Agent | Easy for everyone to understand |
| RAG Assistant with citations | Shows enterprise knowledge-grounding |
| BFSI Maker-Checker / AML | Shows governance/security value |

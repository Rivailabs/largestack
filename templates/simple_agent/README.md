# Simple LARGESTACK Agent

A minimal LARGESTACK agent template.

## Setup
```bash
pip install largestack-agentic-ai
cp .env.example .env
# Edit .env with your OPENAI_API_KEY
```

## Run
```bash
python main.py
# or
largestack run agent.yaml
```

## Customize
Edit `agent.yaml` to change the model, instructions, tools, or guardrails.

# simple_crm

A minimal CRM with contact management and lead scoring.

## Run

```bash
python crm.py
```

## Test

```bash
python -m pytest tests/
```

## Largestack Smoke Test

```bash
python -m pytest tests/test_largestack_features.py -v
```

## Requirements

- Python 3.8+
- largestack library (for smoke test)
- pytest (for tests)

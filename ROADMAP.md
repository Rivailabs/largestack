# LARGESTACK Roadmap

## v0.3 (Q3 2026)
- Postgres-first checkpointer (production-grade durable execution)
- Compiled Cython wheels (linux/macos/windows × py3.11-3.13)
- TypeScript SDK published to npm
- Published benchmarks (τ-bench, GAIA, Ragas, LongMemEval)
- MkDocs site live at docs.largestack.ai

## v0.4 (Q4 2026)
- DSPy-style prompt optimization
- Code-mode agents (write Python instead of JSON tool calls)
- Voice agents (gpt-realtime + SIP)
- Computer use / browser automation
- YAML declarative agents

## v1.0 (Q1 2027)
- Production hardening
- SOC 2 Type II compliance
- Enterprise support tier
- 5+ anchor production users

## Long-term
- Multi-language SDKs (Java, Go, Rust)
- LangGraph Studio-equivalent debugger
- Hosted cloud platform
